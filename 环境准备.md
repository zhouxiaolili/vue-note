# vue 快速原型开发
1. 首先安装全局扩展

   `npm install -g @vue/cli-service-global`

2. 启动开发服务器

   `vue serve ./Hello.vue`

3. 创建项目

   `vue create project-name`
4. 使用插件

   `vue add router`
